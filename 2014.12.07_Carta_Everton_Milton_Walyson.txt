A análise feita com 100000 IDs de partículas, povoadas nas cinco 
estruturas de dados – Listas Encadeadas, Hash com linear probing, 
Hash com listas encadeadas, arvores binarias de pesquisa e arvores AVL - 
nos garantiu uma análise de desempenho detalhada de todas as estruturas 
utilizadas, após os procedimentos de inserção e busca dos IDs nas cinco 
estruturas. Comprovamos que a melhor estrutura de dados para busca de 
IDs é o Hash com Listas Encadeadas, que garantiu uma busca ótima devido 
o seu método de tratamento de colisão, com media aproximada de 1,5 
comparações para cada ID, número muito menor se comparado com qualquer 
uma das outras quatro estruturas.
